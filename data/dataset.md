# Dataset Descriptions and Download Sources

## Alemany-Fin

**Citation:** [@adam2017psychrophilic]

The Alemany-Fin dataset was derived from caudal fin tissue of zebrafish (Danio rerio) and is an scRNA-seq dataset. This dataset was generated using the ScarTrace lineage-tracing technology. The training and test sets were split by donor to evaluate model generalization across independent biological samples. The raw RNA sequencing data are available from the NCBI Gene Expression Omnibus (GEO) database under the accession number GSE102990. The h5ad file can be downloaded from the Cell BLAST data download portal: `https://cblast.gao-lab.org/download`.

## BM-RNA

**Citation:** [@persad2023seacells]

The BM-RNA dataset was derived from T cell depleted bone marrow samples from healthy human (Homo sapiens) donors. It represents the RNA transcriptomic modality of single cell multiome data. This dataset was generated using the 10x Genomics Single Cell Multiome ATAC + Gene Expression platform. The training and test sets were split according to sequencing batches to evaluate model generalization across independent batches. The raw and processed data are available from the NCBI GEO database under accession number GSE200046. Processed h5ad files are also available from Zenodo under DOI 10.5281/zenodo.6383269.

## Bassez

**Citation:** [@bassez2021single]

The Bassez dataset was derived from tumor biopsy samples collected from human (Homo sapiens) breast cancer patients before and after anti PD1 treatment. It is an scRNA-seq dataset generated using the 10x Genomics Chromium platform. The training and test sets followed the original dataset split, in which cancer cells were present only in the test set. The raw sequencing data are available from the EGA database under accession number EGAS00001004809, with dataset accession EGAD00001006608. The processed h5ad file is also available from the scMARK-v2 dataset [@swechha2021scmark] under DOI 10.5281/zenodo.7795653.

## PBMC-Sub

**Citation:** [@hao2021integrated]

The PBMC-Sub dataset was derived from the PBMC seurat v4 dataset, which comprises human (Homo sapiens) peripheral blood mononuclear cells (PBMCs). The dataset was generated using CITE seq technology. In this study, the RNA transcriptomic data from this dataset were used. The training and test sets were split according to sequencing time points to evaluate model generalization across different sequencing times. In the test set, distinct B-cell subtypes were included to evaluate discovery performance on novel yet closely related cell populations. The processed h5ad file can be obtained through the scvi tools dataset interface `scvi.data.pbmc_seurat_v4_cite_seq()`, and is also available from the Figshare scvi tools reproducibility processed data repository: `https://figshare.com/articles/dataset/scvi-tools-reproducibility_processed_data/14374574`.

## TMS-droplet-Thymus

**Citation:** [@tabula2020single]

The TMS-droplet-Thymus dataset was derived from mouse (Mus musculus) thymus tissue data and is a single cell RNA transcriptomic dataset. This dataset is part of the Tabula Muris Senis project and was generated using 10x Genomics droplet based single cell RNA seq technology. The training and test sets were split according to the sex of the mouse samples to evaluate model generalization across sex-specific biological variation. The processed data can be downloaded from Figshare at `https://figshare.com/projects/Tabula_Muris_Senis/64982`.

## BoneMarrowA-BoneMarrowB

**Citation:** [@cusanovich2018single]

The raw data are available from the NCBI GEO database under accession number GSE111586, with the corresponding samples GSM3034622 and GSM3034623. ## Zeisel-2018

**Citation:** [@zeisel2018molecular]

The Zeisel-2018 dataset was derived from the nervous system of mouse (Mus musculus). It is a single-cell RNA sequencing dataset. This study employed 10x Genomics single-cell sequencing technology to profile a large number of individual cells and construct a comprehensive cellular atlas of the mouse nervous system. The training and test sets were separated by developmental stage to evaluate model generalization across stage-specific biological variation. The dataset can be downloaded from the Cell BLAST data download portal: `https://cblast.gao-lab.org/download`.

## HNSCC-RNA

**Citation:** [@rahim2023dynamic]

The HNSCC-RNA dataset was derived from tumor and regional lymph node samples from human (Homo sapiens) patients with head and neck squamous cell carcinoma (HNSCC). It is an scRNA-seq dataset. The training and test sets contained different CD8 T-cell states to evaluate model generalization across distinct cellular states. The raw and processed data are available from the NCBI GEO database under accession number GSE212797.

## HNSCC-ADT

**Citation:** [@rahim2023dynamic]

The HNSCC-ADT dataset was derived from tumor and regional lymph node samples from human (Homo sapiens) patients with head and neck squamous cell carcinoma (HNSCC). It is a single cell surface protein expression dataset. The dataset was generated using CITE seq technology, in which antibody derived tags were used to quantify protein expression at the single cell level. The training and test sets contained different CD8 T-cell states to evaluate model generalization across distinct cellular states. The raw and processed data are available from the NCBI GEO database under accession number GSE212797.
